## Описание (что изучили)

### 1. Структура DOM
- DOM как древовидная структура HTML-документа
- Объект `document` — точка входа в DOM
- Просмотр DOM-дерева в консоли браузера

### 2. Поиск элементов в DOM
- `getElementById()` — поиск по уникальному идентификатору
- `querySelector()` — универсальный поиск по CSS-селекторам
- Разница между методами поиска

### 3. Изменение содержимого и стилей
- `textContent` — изменение текстового содержимого
- Свойство `style` — управление CSS-стилями через JavaScript
- Динамическое применение стилей

### 4. Обработка событий
- `addEventListener()` — назначение обработчиков событий
- Событие `click` — реакция на клик мыши
- Изменение внешнего вида элементов в ответ на действия пользователя

### 5. Работа с формами и вводом данных
- Получение значений из `input` через свойство `value`
- Валидация данных (проверка на пустые значения)
- Динамическое отображение результатов
- Метод `focus()` для установки курсора в поле

### 6. Валидация форм
- Проверка обязательных полей
- Проверка числовых значений на диапазон
- Пользовательские сообщения об ошибках
- `preventDefault()` для отмены стандартной отправки формы
- `form.reset()` для очистки полей

### 7. Динамическое создание элементов
- `createElement()` — создание новых HTML-элементов
- `appendChild()` — добавление элементов в DOM
- Делегирование событий для динамически созданных элементов
- Создание счетчиков и статистики

## Сравнение с C# (WinForms/WPF)

| Концепция | C# (WinForms) | JavaScript (DOM) |
|-----------|---------------|-------------------|
| Найти элемент | `Button btn = this.Controls["myButton"]` | `const btn = document.getElementById("myButton")` |
| Изменить текст | `label1.Text = "Новый текст"` | `label1.textContent = "Новый текст"` |
| Добавить обработчик | `button1.Click += HandleClick` | `button.addEventListener("click", handleClick)` |
| Создать элемент | `Button btn = new Button()` | `const btn = document.createElement("button")` |
| Добавить в контейнер | `panel1.Controls.Add(btn)` | `panel.appendChild(btn)` |
| Скрыть элемент | `button1.Visible = false` | `button.style.display = "none"` |
